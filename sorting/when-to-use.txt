#1 - Sort 10 schools around your house by distance
    Insertion sort - easy to code, fast for small inputs

#2 - eBay sorts listings by the current bid amount
    Radix or counting sort

#3 - Sort scores on ESPN
    Quick sort

#4 - Massive database (can't fit all into memory) needs to sort through past year's users database
    Merge sort - Can't fit it into memory anyway, so no possibility of worst case sort in Quick sort scenario

#5 - Almost sorted Udemy review data needs to update and add 2 new reviews
    Insertion sort

#6 - Temperature records for the past 50 years in Canada
    Radix or counting if no decimal, if not Quick
#7 - Large user name database needs to be sorted. Data is very random
    Quick sort

#8 - Want to teach sorting for the first time
    Bubble sort or Selection sort
